# learning-backbone
This is a demo project to learn backbone

It has 2 parts:

1. The [reference](reference.md)
2. The [code](app/)

### How to run the app

Download the repo
```
$ git clone git@github.com:alexserver/learning-backbone.git
```

Verify you have Node v0.12.x to v4.0.x

Install packages
```
$ npm install
```

Run the app
```
$ npm run start
```
